## ✅ 为什么做这个？

在长期算法教学过程中发现，很多同学：

- 会写代码，但不知道为什么要这么写；
- 学了语法，却写不出结构完整的程序；
- 刷了很多题，依然缺乏“题目结构感”；

  所以想做一套慢慢讲清楚**如何把一个题，从输入读懂，到结构讲透，到代码写出，到输出正确。**，让更多同学感受到算法的秩序之美。
>
